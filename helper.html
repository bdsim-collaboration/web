BDSIM helper/support packages

Pybdsim

A python pacakge to help generating BDSIM input, so gmad files, fields etc. It is the foundation for converters from other beam codes to BDSIM.

Pymadx

MAD-X a very well established beam optics code from CERN. To be able to convert a MAD-X lattice to BDSIM, MAD-X output needs to be loaded. This is performed by pymadx

Pyg4ometry

is a large Python package to load, manipulate, edit, display and composite geometry for Geant4 (GDML) and FLUKA. In essence it wraps the Geant4 C++ geometry code or FLUKA input cards with Python, so users can programatically generate geometry
