Docker images

Docker images are avaible to building BDSIM. The base OS images are

  • centos 8
  • alma 9
  • Ubuntu 20 LTS
  • Ubuntu 22 LTS
  • Ubuntu 24 LTS

For each of these operating systems multiple Geant4 versions are built

  • Geant4 10.7
  • Geant4 11.0
  • Geant4 11.1
  • Geant4 11.2
  • Geant4 11.3
