Getting started

Complete environment

  • If using conda create/activate an environment
  • Install or obtain BDSIM (from conda, docker, from source or CVMFS)
  • Install ROOT 
    • (brew/yum/apt/conda) install ROOT
  • Install helper packages 
    • pip install --name pybdsim pandas
  • Test 
    terminal> bdsim --file={{GMAD file}} --outfile={{OUTPUT file}} --ngenerate=100 --batch
    teriminal>ipython --pylab
    python> import pybdsim
    python> d = pybdsim.DataPandas..BDSIMOutput({OUTPUT file}})
  • If your are using conda BDSIM you'll need to set the following variables 
    export BDSIM={{ location of BDSIM dir }}
    export ROOT_INCLUDE_PATH=${ROOT_INCLUDE_PATH}:$BDSIM/include/bdsim/:$BDSIM/include/bdsim/analysis/:$BDSIM/include/bdsim/parser/

Conda

Installing BDSIM from Conda

  • Install miniconda for your system
  • Create conda environment for BDSIM, e.g. 
    conda create --name bdsim
  • Activate conda bdsim environment e.g. 
    conda activate bdsim
  • Install BDSIM from conda bdsim-g4 package 
    conda install bdsim-g4 -c conda-forge
  • Run BDSIM 
    bdsim --file={{GMAD FILE}}

Docker

  • Install Docker desktop for your OS
  • Pull docker image 
    docker pull bdsim/alma9-g4.11.3-bdsim:develop
  • Determine your IP address 
    ifconfig/ipconfig
  • Enable XWindows access in the host 
    xhost +
  • Start container 
    docker run -ti  bdsim/alma9-g4.11.3-bdsim:develop
  • Set display variable in container 
    export DISPLAY={{IP ADDRESS}}:0
  • Run BDSIM 
    bdsim --file={{GMAD FILE}}

Mac XQuartz

To enable OpenGL over Xwindows for mac
  • Download and install XQuartz
  • Enable OpenGL in XQuartz 
    defaults write org.xquartz.X11 enable_iglx -bool true
  • restart xquartz

VNC

VNC XServer. Occasionally OpenGL for Geant4 visualisation fails. 
$host> xhost +
$host> docker run -p 5900:5900 -ti alma9-g4.11.3-bdsim:develop
$container> Xvfb :1 -screen 0 1024x768x24 &
$container> export DISPLAY=:1
$container> x11vnc -display :1
$container> fvwm &
$container> xterm &

Compilation

Requires various dependencies (CLHEP, Geant4, Xerces, Qt) 

  • terminal> git clone git@github.com:bdsim-collaboration/bdsim.git
  • terminal> mkdir build
  • terminal> mkdir install
  • terminal> cd build
  • terminal> cmake ../bdsim --DCMAKE_INSTALLPREFIX=../install
  • terminal> make install
  • terminal> source ../instal/bin/bdsim.sh
